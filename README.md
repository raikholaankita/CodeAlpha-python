# Hangman Game - CodeAlpha Task 1

A classic word-guessing game built with Python for the CodeAlpha Internship.

## 🎮 Features
- Random word selection from a predefined list
- Player gets 6 lives to guess the word
- Displays guessed letters and remaining lives
- Prevents repeated letter guesses
- Clear win/lose messages

## 🧠 Concepts Used
`random` module, `while` loops, `if-elif-else` conditions, `lists`, string manipulation, `input()` and `print()`

## ▶️ How to Run
```bash
python hangman.py
```
