## 📋 Tasks Completed

### 1. Hangman Game `hangman.py`
A classic word-guessing game built with Python.

**🎮 Features:**
- Random word selection from a predefined list
- Player gets 6 lives to guess the word
- Displays guessed letters and remaining lives
- Prevents repeated letter guesses
- Clear win/lose messages

**▶️ How to Run:**
```bash
python hangman.py
```
### 2. Basic Chatbot `chatbot.py`
A simple rule-based chatbot that responds to predefined user inputs.

**🤖 Features:**
- Responds to greetings: `hello`, `hi`, `hii`, `hey`, `namaste`
- Answers basic questions like `how are you` and `your name`
- Exit commands: `bye`, `exit`
- Handles extra spaces and case-insensitivity
- Default reply for unknown inputs

**🧠 Core Concepts:** `functions`, `while` loop, `if-elif-else`, `lists`, string methods `.lower()` and `.strip()`

**▶️ How to Run:**
```bash
python chatbot.py
```
